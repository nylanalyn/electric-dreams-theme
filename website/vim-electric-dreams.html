Electric Dreams for Vim

Bold neon colors for your editor

Features

  • High contrast design with dramatic color pops
  • True color (24-bit) support
  • Full TreeSitter syntax highlighting support
  • LSP diagnostics integration
  • Optimized for Neovim and Vim 8+
  • Language-specific highlighting (JS/TS, Python, HTML/CSS, Markdown, etc.)
  • Git integration (GitGutter, Signify)

Installation

Using vim-plug

Add this to your ~/.vimrc or ~/.config/nvim/init.vim:

Plug 'nylanalyn/electric-dreams-theme'

Then run :PlugInstall

Using Vundle

Add this to your ~/.vimrc:

Plugin 'nylanalyn/electric-dreams-theme'

Then run :PluginInstall

Using Pathogen

cd ~/.vim/bundle
git clone https://github.com/nylanalyn/electric-dreams-theme.git

Using Vim 8's Native Package Manager

mkdir -p ~/.vim/pack/themes/start
cd ~/.vim/pack/themes/start
git clone https://github.com/nylanalyn/electric-dreams-theme.git

Manual Installation

Download the colorscheme file and copy it to your colors directory:

mkdir -p ~/.vim/colors
curl -o ~/.vim/colors/electric_dreams.vim \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/vim/colors/electric_dreams.vim

Activation

Add this to your ~/.vimrc or ~/.config/nvim/init.vim:

" Enable 24-bit true colors
if (has("termguicolors"))
  set termguicolors
endif

" Set the colorscheme
colorscheme electric_dreams

For Neovim users using Lua configuration (~/.config/nvim/init.lua):

-- Enable true colors
vim.opt.termguicolors = true

-- Set the colorscheme
vim.cmd([[colorscheme electric_dreams]])
