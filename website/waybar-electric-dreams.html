Installation

Quick Install

# Download the theme
mkdir -p ~/.config/waybar/themes
curl -o ~/.config/waybar/themes/electric-dreams.css \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/waybar/style.css

# Backup your current style
cp ~/.config/waybar/style.css ~/.config/waybar/style.css.backup

# Apply the theme
cp ~/.config/waybar/themes/electric-dreams.css ~/.config/waybar/style.css

# Restart waybar
killall waybar && waybar &

Alternative: Use with @import

Keep your existing styles and import Electric Dreams:

# Download to themes directory
mkdir -p ~/.config/waybar/themes
curl -o ~/.config/waybar/themes/electric-dreams.css \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/waybar/style.css

Add this line to the top of your ~/.config/waybar/style.css:

@import "themes/electric-dreams.css";

NixOS / Home Manager

Add to your Home Manager configuration:

programs.waybar = {
  enable = true;
  style = builtins.readFile ./waybar-electric-dreams/style.css;
};

Customization

Tweak these common settings by editing the CSS file:

Adjust Transparency

window#waybar {
    background-color: rgba(20, 20, 25, 0.95);  /* Change 0.95 to desired opacity */
}

Change Module Colors

#cpu {
    color: #9c27b0;  /* Change to any color you prefer */
    background-color: rgba(156, 39, 176, 0.1);
}

Modify Border

window#waybar {
    border-bottom: 2px solid #00d9ff;  /* Change color or size */
}

Font Size

* {
    font-size: 13px;  /* Adjust to your preference */
}

  • True color terminal support

Troubleshooting

Waybar not updating?

# Restart waybar
killall waybar && waybar &

# Or reload config (if supported)
pkill -SIGUSR2 waybar

Font not displaying correctly?

# Install JetBrains Mono Nerd Font
# Arch
sudo pacman -S ttf-jetbrains-mono-nerd

# Or edit style.css to use your preferred font
font-family: 'Your Font Name', monospace;

Colors look wrong?

Make sure your compositor supports transparency and your waybar config doesn't override the theme.
