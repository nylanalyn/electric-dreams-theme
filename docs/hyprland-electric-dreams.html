Installing Hyprland Theme

Option 1: Source the theme (Recommended)

# Create themes directory
mkdir -p ~/.config/hypr/themes

# Download the theme
cd ~/.config/hypr/themes
curl -O https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/hyprland/hyprland.conf

# Rename it
mv hyprland.conf electric-dreams.conf

Then add this to your ~/.config/hypr/hyprland.conf:

source = ~/.config/hypr/themes/electric-dreams.conf

Option 2: Direct integration

Copy the contents of hyprland.conf directly into your main Hyprland config.

Installing Hyprlock Theme

Step 1: Download the configuration

# Direct replacement
curl -o ~/.config/hypr/hyprlock.conf \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/hyprland/hyprlock.conf

Or keep it separate:

# As a theme file
mkdir -p ~/.config/hypr/themes
curl -o ~/.config/hypr/themes/electric-dreams-lock.conf \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/hyprland/hyprlock.conf

# Then source it in your hyprlock.conf
echo "source = ~/.config/hypr/themes/electric-dreams-lock.conf" >> ~/.config/hypr/hyprlock.conf

Step 2: Install required font

# Arch/Manjaro
sudo pacman -S ttf-jetbrains-mono-nerd

# Ubuntu/Debian
sudo apt install fonts-jetbrains-mono

Step 3: Install jq (for keyboard layout display)

# Arch/Manjaro
sudo pacman -S jq

# Ubuntu/Debian
sudo apt install jq

Customization

Adjust these common settings to match your preferences:

Border Size & Gaps

general {
    gaps_in = 5       # Inner gaps (default: 5)
    gaps_out = 10     # Outer gaps (default: 10)
    border_size = 2   # Border width (default: 2)
}

Animation Speed

# Make animations faster (lower duration)
animation = windows, 1, 3, overshot, slide

# Make animations slower (higher duration)
animation = windows, 1, 7, overshot, slide

Custom App Border Colors

# Add these to hyprland.conf
windowrulev2 = bordercolor rgba(1de9b6ee), class:(firefox)
windowrulev2 = bordercolor rgba(ff006eee), class:(kitty)
windowrulev2 = bordercolor rgba(9c27b0ee), class:(discord)
