Electric Dreams for GTK

Neon CSS overrides that bring cyan glow and magenta accents to GTK 3 & 4 apps.

Features

Dark Surfaces

Windows, dialogs, and popovers use deep black backgrounds with subtle borders.

Accent Buttons

Suggested and destructive buttons glow in hot magenta with cyan focus rings.

Rounded Controls

Entries, cards, progress bars, and switches pick up 8–12px rounding to match Hyprland.

Neon Focus

Cyan border and drop shadow highlight focused controls for fast navigation.

Installation

GTK 4

mkdir -p ~/.config/gtk-4.0
curl -o ~/.config/gtk-4.0/gtk.css \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/gtk/gtk.css

GTK 3

mkdir -p ~/.config/gtk-3.0
curl -o ~/.config/gtk-3.0/gtk.css \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/gtk/gtk.css

Restart your session or relaunch GTK apps to apply the changes.

Customization

@define-color electric-accent #1DE9B6;  /* swap to turquoise */

button.suggested-action,
button.destructive-action,
.accent {
  background-color: @electric-accent;
  border-color: @electric-accent;
}

Lower the cyan glow by reducing the box-shadow alpha, or tighten corners by changing border-radius values.

Recommended Pairings

  • Adwaita Variant: Install adw-gtk3 for a modern base then add this override.
  • Icons: Papirus-Nord or Tela icons complement the neon palette.
  • Mako / Rofi: Combine with the notification and launcher ports for visual consistency.

Requirements

  • GTK-based desktop environment or apps (GNOME, XFCE, etc.)
  • Writable ~/.config/gtk-3.0 and gtk-4.0 directories
  • Optional: JetBrainsMono Nerd Font for UI consistency
