Electric Dreams for Starship

Shell prompt palette with cyan arrow glyphs and magenta accents.

Features

Custom Palette

Defines an Electric Dreams palette for use across modules.

Cyan Prompt Glyphs

Statement arrows with bold cyan foreground and red error fallback.

Right-Side Clock

Time module floats on the right, mirroring tmux status bar.

Language Colors

Node, Python, and Rust modules adopt turquoise, magenta, and purple accents.

Installation

curl -o ~/.config/starship.toml \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/starship/starship.toml

Enable Starship in your shell (example for zsh):

echo 'eval "$(starship init zsh)"' >> ~/.zshrc
Download README

Format Overview

format = """
[$directory]($style)[$git_branch$git_status]($style)$nodejs$rust$python$character
"""
right_format = """$time"""
palette = "electric_dreams"

Customization

# Make the prompt multi-line
add_newline = true
format = """
$directory$git_branch$git_status
$nodejs$rust$python$character
"""

# Swap prompt symbol
[character]
success_symbol = "[➜](bold cyan)"

Disable the right-side clock by removing right_format, or add more modules (Docker, AWS, etc.) using the Electric Dreams palette.

Requirements

  • Starship 1.14+
  • Shell initialization (zsh, bash, fish, PowerShell…)
  • JetBrainsMono Nerd Font for glyph consistency
