Installation

Quick Install

# Create themes directory
mkdir -p ~/.config/kitty/themes

# Download the theme
curl -o ~/.config/kitty/themes/electric-dreams.conf \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/kitty/electric-dreams.conf

# Add to your kitty.conf
echo "include themes/electric-dreams.conf" >> ~/.config/kitty/kitty.conf

# Reload Kitty (Ctrl+Shift+F5)
üí° Tip: Press Ctrl+Shift+F5 in Kitty to reload the configuration without restarting.

Recommended Settings

Add these to your ~/.config/kitty/kitty.conf for the complete experience:

Font

font_family      JetBrainsMono Nerd Font
bold_font        JetBrainsMono Nerd Font Bold
italic_font      JetBrainsMono Nerd Font Italic
font_size        11.0

Window Style

window_padding_width  8
window_border_width   2.0
draw_minimal_borders  yes

# Optional: transparency and blur
background_opacity    0.95
background_blur       32

Tab Bar

tab_bar_edge          top
tab_bar_style         powerline
tab_powerline_style   slanted

Cursor

cursor_shape          beam
cursor_beam_thickness 2.0
cursor_blink_interval 0

Customization

Adjust Opacity

# More transparent
background_opacity 0.85

# Fully opaque
background_opacity 1.0

Change Cursor Style

# Block cursor
cursor_shape block

# Underline cursor
cursor_shape underline

Move Tab Bar

# Bottom tab bar
tab_bar_edge bottom

# Hidden tab bar
tab_bar_style hidden
