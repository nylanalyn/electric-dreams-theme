Ports & Applications

Get the Electric Dreams theme for your favorite applications:

Vim / Neovim

Full-featured colorscheme with TreeSitter and LSP support

Plug 'nylanalyn/vim-electric-dreams'

Hyprland

Neon window manager theme with glowing borders and smooth animations

source = ~/.config/hypr/themes/electric-dreams.conf

Hyprlock

Matching lock screen with neon rings and animated displays

~/.config/hypr/hyprlock.conf

Kitty Terminal

True color terminal theme with electric cyan cursor

include themes/electric-dreams.conf

Visual Studio Code

Complete UI and syntax theme for VSCode and VSCodium

Install from Marketplace

Firefox

Browser theme with cyan tab indicators and neon accents

Install from Firefox Add-ons

Waybar

Status bar theme with color-coded modules and glowing effects

~/.config/waybar/style.css

tmux

Terminal multiplexer theme with cyan borders and magenta highlights

set -g @plugin 'nylanalyn/electric-dreams-theme'

Neomutt

Terminal email client theme with color-coded messages and neon accents

source ~/.config/neomutt/themes/electric-dreams.muttrc
