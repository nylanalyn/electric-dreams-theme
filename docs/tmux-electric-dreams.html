Quick Install

Download and apply the theme in one command:

# Create theme directory
mkdir -p ~/.config/tmux/themes

# Download theme file
curl -o ~/.config/tmux/themes/electric-dreams.conf \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/tmux/electric-dreams.conf

# Add to tmux config
echo "source-file ~/.config/tmux/themes/electric-dreams.conf" >> ~/.tmux.conf

# Reload tmux
tmux source-file ~/.tmux.conf

That's it! The theme should now be active.

Manual Installation

Step 1: Download the theme file

Visit the theme file on GitHub and copy its contents.

Step 2: Create the theme file

mkdir -p ~/.config/tmux/themes
nano ~/.config/tmux/themes/electric-dreams.conf

Paste the theme contents and save the file.

Step 3: Source in your tmux config

Add to ~/.tmux.conf:

source-file ~/.config/tmux/themes/electric-dreams.conf

Step 4: Reload tmux

tmux source-file ~/.tmux.conf

Customization

Add these customizations after sourcing the theme in your ~/.tmux.conf:

Move Status Bar to Top

set -g status-position top

Adjust Status Bar Length

set -g status-left-length 50
set -g status-right-length 150

Custom Status Right

# Add custom modules
set -g status-right "#[fg=#9c27b0] CPU #{cpu_percentage} #[fg=#e91e63] RAM #{ram_percentage} #[fg=#00d9ff,bold] %H:%M "

Change Pane Border Style

# Heavy borders (requires tmux 3.2+)
set -g pane-border-lines heavy

# Or use single/double/number
set -g pane-border-lines double

Override Colors

# Use a different active window color
setw -g window-status-current-style "bg=#1de9b6,fg=#0a0a0f,bold"

Troubleshooting

Theme not loading?

# Check if tmux config has errors
tmux source ~/.tmux.conf

# Verify file path
ls -la ~/.config/tmux/themes/electric-dreams/

Colors look wrong?

Make sure your terminal supports true color. Add this to your shell config:

# For bash/zsh
export TERM=xterm-256color

Or set it in tmux.conf:

set -g default-terminal "tmux-256color"
set -ga terminal-overrides ",*256col*:Tc"

TPM not working?

# Make sure TPM is initialized at the bottom of ~/.tmux.conf
run '~/.tmux/plugins/tpm/tpm'

# Then reload and install
tmux source ~/.tmux.conf
~/.tmux/plugins/tpm/bin/install_plugins
