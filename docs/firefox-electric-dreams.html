Installation

Quick Install

1
Visit Firefox Add-ons
Navigate to the Electric Dreams Neon theme page
2
Click "Add to Firefox"
The theme will download automatically
3
Confirm Installation
Theme applies immediately - enjoy!

Manual Installation (For Testing)

# Navigate to about:debugging in Firefox
# Click "This Firefox"
# Click "Load Temporary Add-on"
# Select the manifest.json file from the theme folder

Note: Temporary installations are removed when Firefox closes.

Advanced Customization

Further customize Firefox with userChrome.css for advanced theming.

Enable userChrome.css

1
Navigate to about:config in Firefox
2
Search for toolkit.legacyUserProfileCustomizations.stylesheets
3
Set to true and restart Firefox

Example: Rounded Tabs

/* userChrome.css */
.tab-background {
  border-radius: 8px 8px 0 0 !important;
}

.tab-line {
  height: 3px !important;
}
