Electric Dreams for mako

Wayland notifications with cyan borders, magenta progress bars, and translucent glow.

Preview

Rounded 12px cards, cyan borders, and accent overrides for Spotify, Discord, and Thunderbird keep notifications on-theme.

Features

Semi-Transparent Cards

Notifications float above Hyprland blur without losing readability.

Urgency Colors

Green borders for low urgency, cyan for normal, magenta/red for high priority.

Large Icon Support

64px icons look crisp for media controls or chat apps.

App Overrides

Spotify, Discord, and Thunderbird get dedicated border colors for quick identification.

Installation

mkdir -p ~/.config/mako
curl -o ~/.config/mako/config \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/mako/electric-dreams.conf

Reload mako:

killall mako && mako &
Download README

Hyprland Auto-Start

# ~/.config/hypr/hyprland.conf
exec-once = mako

Customization

# Longer timeout for normal urgency
[urgency=normal]
default-timeout=7000

# Smaller icons
[global]
max-icon-size=48

Add more app-specific border colors by appending new [app-name=XYZ] blocks to the config.

Requirements

  • Mako notification daemon
  • Wayland compositor (Hyprland, sway, river, etc.)
  • Optional: JetBrainsMono Nerd Font
