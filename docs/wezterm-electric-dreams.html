Electric Dreams for WezTerm

Neon color scheme with cyan cursor glow and magenta tab accents.

Features

Color Scheme

Registers an Electric Dreams palette you can reuse across profiles.

Tab Styling

Flat tabs with cyan active states and magenta hover glow to match Waybar.

Pane Dimming

Inactive panes subtly dim to keep focus on the active window.

Window Padding

Soft 8px padding looks great with transparent backgrounds and blur.

Installation

Drop-in module

mkdir -p ~/.config/wezterm
curl -o ~/.config/wezterm/electric_dreams.lua \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/wezterm/electric_dreams.lua

Import the module in wezterm.lua:

local wezterm = require("wezterm")
local electric_dreams = require("electric_dreams")

return wezterm.extend_config(electric_dreams)

Manual merge

If you already return a table, pull the fields into your config:

local electric_dreams = require("electric_dreams")

return {
  color_schemes = electric_dreams.color_schemes,
  color_scheme = electric_dreams.color_scheme,
  window_padding = electric_dreams.window_padding,
  inactive_pane_hsb = electric_dreams.inactive_pane_hsb,
  use_fancy_tab_bar = electric_dreams.use_fancy_tab_bar,
}

Customization

-- Prefer the default tab bar?
use_fancy_tab_bar = true

-- More contrast on inactive panes
inactive_pane_hsb = { saturation = 0.75, brightness = 0.65 }

-- Swap cursor color
colors.cursor_bg = "#FF006E"
colors.cursor_border = "#FF006E"

All palette values are defined at the top of electric_dreams.lua, so you can tweak accents without touching the rest of the config.

Requirements

  • WezTerm 20230408 or newer
  • True color (24-bit) terminal support
  • Optional: JetBrainsMono Nerd Font for consistent glyphs
