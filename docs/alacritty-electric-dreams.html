Electric Dreams for Alacritty

GPU terminal palette with cyan cursor glow and magenta highlights.

Features

Neon Palette

Primary and bright ANSI colors mapped to Electric Dreams cyan, magenta, and purple hues.

Cursor Glow

Beam cursor in electric cyan with contrasting black text for readability.

Dynamic Padding

Soft inner padding keeps terminals centered when transparency is enabled.

Notification Pulse

Optional magenta visual bell to keep alerts on-theme without being intrusive.

Installation

Quick import

mkdir -p ~/.config/alacritty/colors
curl -o ~/.config/alacritty/colors/electric-dreams.yml \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/alacritty/electric-dreams.yml

Add the theme to alacritty.yml:

import:
  - ~/.config/alacritty/colors/electric-dreams.yml

Manual merge

Copy the contents of electric-dreams.yml into your existing colors: block if you prefer a single file configuration.

Color Palette

ANSI Hex Usage
Black		 #0A0A0F Window background
Electric Cyan		 #00D9FF Cursor, active selections
Hot Magenta		 #FF006E Bright yellow slot (contrast highlight)
Deep Purple		 #9C27B0 Magenta slot, prompt glyphs
Turquoise		 #1DE9B6 Green slot & hints
Light Lavender		 #D0D0E0 Foreground text

Customization

# Change cursor style
cursor:
  style:
    shape: Block
    blinking: Off

# Softer padding
window:
  padding:
    x: 4
    y: 4

Disable the magenta visual bell pulse by removing the bell section or setting duration: 0.

Requirements

  • Alacritty 0.12+
  • GPU with 24-bit color support
  • JetBrainsMono Nerd Font (optional)
