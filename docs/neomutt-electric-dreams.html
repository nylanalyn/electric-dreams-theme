Installation

Quick Install (Recommended)
Manual Copy-Paste

Quick Install

Download and apply the theme in one command:

# Create theme directory
mkdir -p ~/.config/neomutt/themes

# Download theme file
curl -o ~/.config/neomutt/themes/electric-dreams.muttrc \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/neomutt/electric-dreams.muttrc

# Add to neomutt config
echo "source ~/.config/neomutt/themes/electric-dreams.muttrc" >> ~/.config/neomutt/neomuttrc

# Restart Neomutt or reload with :source

That's it! The theme should now be active.

Manual Installation

Step 1: Download the theme file

Visit the theme file on GitHub and copy its contents.

Step 2: Create the theme file

mkdir -p ~/.config/neomutt/themes
nano ~/.config/neomutt/themes/electric-dreams.muttrc

Paste the theme contents and save the file.

Step 3: Source in your neomutt config

Add to ~/.config/neomutt/neomuttrc or ~/.neomuttrc:

source ~/.config/neomutt/themes/electric-dreams.muttrc

Step 4: Reload Neomutt

Restart Neomutt or reload with:

:source ~/.config/neomutt/neomuttrc

Customization

Override specific colors by adding additional color commands after sourcing the theme in your config file:

Use Different Indicator Color

source ~/.config/neomutt/themes/electric-dreams.muttrc

# Override the indicator color to use hot magenta
color indicator color232 color213

Change New Message Color

# Make new messages use turquoise instead of cyan
color index color85 color232 ~N

Modify Quoted Text Colors

# Use different colors for quote levels
color quoted  color199 color232  # Pink for level 1
color quoted1 color141 color232  # Purple for level 2

Recommended Neomutt Settings

For the best experience with Electric Dreams:

Enable Sidebar

set sidebar_visible = yes
set sidebar_width = 25
set sidebar_format = "%D%* %?N?%N/?%S"
set sidebar_new_mail_only = no

Better Threading

set sort = threads
set sort_aux = reverse-last-date-received

Improved Index Format

set index_format = "%4C %Z %{%b %d} %-15.15L (%?l?%4l&%4c?) %s"

Troubleshooting

Theme not loading?

# Check if config has errors
neomutt -n

# Verify file path
ls -la ~/.config/neomutt/themes/electric-dreams.muttrc

Colors look wrong?

Make sure your terminal supports 256 colors or true color. Check with:

# Test 256 color support
curl -s https://gist.githubusercontent.com/lifepillar/09a44b8cf0f9397465614e622979107f/raw/24-bit-color.sh | bash

Set your terminal properly:

# For bash/zsh
export TERM=xterm-256color

Sidebar not showing?

Make sure sidebar is enabled:

set sidebar_visible = yes
