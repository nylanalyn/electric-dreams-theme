Electric Dreams for Rofi

Transparent application launcher with cyan prompt and magenta selection glow.

Live Preview

Electric cyan prompt, hot magenta selection background, and deep violet surface highlight the apps list.

Features

Transparent Window

Real transparency with blur-friendly background to match Hyprland.

Cyan Prompt

Accent underline and prompt colour make the search input pop.

Icon Friendly

Supports app icons and multiple modi—drun, run, and window.

Rounded Corners

8px card styling mirrors the Waybar and GTK overrides.

Installation

mkdir -p ~/.config/rofi/themes
curl -o ~/.config/rofi/themes/electric-dreams.rasi \
  https://raw.githubusercontent.com/nylanalyn/electric-dreams-theme/main/ports/rofi/electric-dreams.rasi

Launch Rofi with the theme:

rofi -show drun -theme ~/.config/rofi/themes/electric-dreams.rasi

Persist the theme

mkdir -p ~/.config/rofi
ln -sf ~/.config/rofi/themes/electric-dreams.rasi ~/.config/rofi/electric-dreams.rasi

# ~/.config/rofi/config.rasi
configuration {
  theme: "electric-dreams";
}
Download README

Hyprland Keybind

# ~/.config/hypr/hyprland.conf
bind = $mod, D, exec, rofi -show drun -theme ~/.config/rofi/themes/electric-dreams.rasi

Customization

# Adjust width
window {
  width: 680px;
}

# More rows
listview {
  lines: 10;
}

Update font family by editing the prompt and entry blocks (JetBrainsMono is used by default).

Requirements

  • Rofi 1.7+ or rofi-wayland
  • Compositor with real transparency for the glow effect
  • JetBrainsMono Nerd Font (optional)
